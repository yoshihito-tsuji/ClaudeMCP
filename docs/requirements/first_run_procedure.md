# 初回検証手順 (macOS)

## 目的

- 撮影（see）と画像認識を最優先で確認する
- 次に、パン・チルトで視野変更ができることを確認する

## 前提

- `docs/requirements/first_run_checklist.md` が完了している
- Claude Code で wifi-cam-mcp が有効化されている

## 手順

1. wifi-cam-mcp を起動する

```bash
cd wifi-cam-mcp
uv run wifi-cam-mcp
```

2. Claude Code に「今何が見える？」と依頼する

3. 返ってきた画像と説明が実際の部屋と整合するか確認する

4. Claude Code に「左を見て」と依頼する

5. 再度「今何が見える？」と依頼し、視点が変わったことを確認する

6. 必要に応じて「右を見る」「上を見る」「下を見る」も同様に確認する

## 判定基準

- `see` で最新の画像が返る
- 画像の説明が現実と大きく乖離していない
- パン・チルト後に視点の変化が確認できる

## トラブル時の確認ポイント

- カメラとMacが同一ネットワークか
- `.env` のIPとアカウント情報が正しいか
- ルーターのIP固定が有効か
- Tapoアプリ側でカメラアカウントが有効か
