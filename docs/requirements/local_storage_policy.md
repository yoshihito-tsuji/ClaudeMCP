# ローカル保存の簡易運用ルール

## 基本方針

- 外部送信は行わない
- 画像保存はローカルのみ許可する
- 保存は必須ではなく、必要時のみ行う

## 保存先（暫定）

- 推奨: `~/Pictures/claude_mcp/`
- 別の場所に保存する場合は、Git管理対象外とする

## ファイル命名（例）

- 例: `capture_YYYYMMDD_HHMMSS.jpg`

## 削除方針

- 容量を圧迫し始めた時点で削除する
- 継続保存が不要になった段階でフォルダごと削除する

## 設定方法（必要時のみ）

- `.env` で `CAPTURE_DIR` を指定する
- `wifi-cam-mcp/.env.example` を参考に設定する
